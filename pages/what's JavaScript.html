What is JavaScript ?

JavaScript (officially call ECMAScript) is the most popular programming language used in development web. it is a high level, Interpreted, dynamic typing and object oriented programming language support imperative and functional style released in 1995 by Brendan Eich. it's one of Standard Web technologie alongside HTML and CSS.

  • HTML : is a markup language allow us to make structure of our web page by defining headings, paragraphs, data table, list, embedded images and videos.
  • CSS : is a rules stylesheet language used to apply styling to web page contents like set color to paragraphs, change size and form of fonts, add bordure and background color etc.
what can it to do ?
we can used JS to make interactive web pages that can update dynamically its content as a respond to deferents event like mouse click, scrolling, button clicked by user, for example a user click a button that trigger JavaScript code wich send a request to server to get some resources: JSON file or image or simply another web page
we can also use JS to manipulate element of web page for example change its content(text, image, and videos) or change style of that content like text color, background color, width and height of paragraphs. you can also create a new element add some content to it and put it wherever you want in your web page using something call DOM (acronym of document object model) so you can do all that on the fly using JS. we will discuss in our tutorial how to do all those stuffs using a real examples.
