Introducing to programming

before you go through javaScript you chould know some important keys related to programing and that's why we are here! to teach you the basics of programming and give you started that make you engaged with upcoming lessons. if you are come in with some acquired knowledge from other programing languages like C/C++,Java, python or PHP, etc. and you want to learn javaScript then go to our first lesson here. otherwise take some minutes to read this useful introduction.

1. programming's meaning

A programing as a concept related to program, you probably ask what's a program? well a program is one or more instructions which a computer could execute its. usually written in programming language using a specific software known as IDE (integrated development environment)
each instruction in a program wich we called statement can do a specific actions. for instance:

  • a statement that tell computer to do a mathematics operations like : addition, soustraction, multiplication and division and store result in a variable.
  • a statement that tell computer to write a new value into memory or to read a value that's previously written from memory.
  • a statement that tell computer to open some resources like a file, connect to a database.
  • a statement that tell computer to write some data to a opened file/database or read the content of that file or delete it permanently.
these are the most common task that a computer can did. with our course you will be familiar with those statements.
