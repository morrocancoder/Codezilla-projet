What's JavaScript ?

JavaScript is the most popular programming language used in development web. it is a high level, Interpreted, dynamic, object oriented programming language support procedural and functional style released in 1995 by(...) it's become one of standard web beside HTML and CSS. especially used to make interactive web page that can change its content as a respond to deferents event like mouse click, scrolling, button clicked by user, etc.
you can also use JS to manipulate element of web page for example change its content(text, image, and videos) or change style of that content like text color, background color, width and height of paragraphs. you can also create a new element and put it wherever you want in your web page using something call DOM (acronym of document object model) we will discuss later how to do all those stuffs using a real examples.

KEY TAKEAWAYS

JavaScript has following futures :

  • Procedural programming
  • Fonctional programming
  • Objet Oriented programming
  • Supported by all traditional browser (Firefox, chrome, IE,Opera)
